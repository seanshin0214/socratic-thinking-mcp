# Think Bubbles (Mind Mapping) - 마인드맵 기법

## 개요
중심 주제에서 방사형으로 아이디어를 확장해 나가는 시각적 사고 도구입니다. 뇌의 자연스러운 연상 작용을 활용합니다.

## 핵심 원리
- 뇌는 비선형적으로 사고함
- 시각화가 기억과 창의성을 촉진
- 연상 작용으로 아이디어 확장

## 적용 단계
1. **중심 주제 배치**: 종이 중앙에 핵심 주제 작성
2. **주요 가지 추가**: 주제와 관련된 주요 카테고리를 가지로 연결
3. **세부 가지 확장**: 각 가지에서 세부 아이디어로 확장
4. **연결 발견**: 다른 가지들 간의 관계 탐색
5. **색상/이미지 추가**: 시각적 요소로 기억력 강화

## 마인드맵 규칙
- 중심에서 바깥으로 확장
- 가지당 하나의 키워드
- 곡선 사용 (직선보다 뇌에 친화적)
- 색상으로 구분
- 이미지와 아이콘 활용

## 활용 예시
```
                    ┌─ 온라인 광고
        ┌─ 마케팅 ──┼─ SNS
        │           └─ 이벤트
        │
신제품 ─┼─ 개발 ────┬─ 디자인
출시    │           └─ 기능
        │
        └─ 유통 ────┬─ 오프라인
                    └─ 온라인
```
