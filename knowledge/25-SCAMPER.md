# SCAMPER - 체계적 발상 전환 기법

## 개요
7가지 질문 프레임워크를 통해 기존 아이디어나 제품을 체계적으로 변형하는 창의적 사고 기법입니다.

## SCAMPER 7가지 원리

### S - Substitute (대체)
"무엇을 다른 것으로 대체할 수 있는가?"
- 재료, 구성요소, 사람, 프로세스 대체
- 예: 설탕 → 스테비아, 플라스틱 → 생분해성 소재

### C - Combine (결합)
"무엇을 합칠 수 있는가?"
- 기능, 제품, 서비스, 아이디어 통합
- 예: 전화 + 카메라 + 컴퓨터 = 스마트폰

### A - Adapt (적응)
"다른 곳에서 무엇을 가져와 적용할 수 있는가?"
- 다른 산업, 자연, 역사에서 영감
- 예: 벨크로 = 도꼬마리 씨앗에서 영감

### M - Modify/Magnify (수정/확대)
"무엇을 크게, 작게, 강하게, 약하게 할 수 있는가?"
- 크기, 형태, 색상, 빈도 변경
- 예: 미니 쿠퍼, 점보 사이즈

### P - Put to other uses (다른 용도)
"다른 방식으로 사용할 수 있는가?"
- 새로운 시장, 새로운 용도 발견
- 예: 베이킹소다 → 탈취제, 세정제

### E - Eliminate (제거)
"무엇을 없앨 수 있는가?"
- 불필요한 요소, 단계, 기능 제거
- 예: 무선 이어폰 (케이블 제거)

### R - Reverse/Rearrange (역전/재배열)
"순서나 구조를 바꿀 수 있는가?"
- 순서, 레이아웃, 역할 변경
- 예: 뷔페식 레스토랑 (손님이 직접 선택)

## 실전 적용 템플릿

- 마케팅 전략 수립
