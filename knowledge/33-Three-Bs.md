# Three B's (Incubation) - 인큐베이션 기법

## 개요
문제를 의식에서 놓아주고 잠재의식이 처리하도록 하는 기법입니다. 세 가지 B(Bed, Bath, Bus)는 통찰이 자주 일어나는 장소를 나타냅니다.

## 핵심 원리
- 의식적 노력 중단 시 잠재의식 작동
- 확산적 사고가 새로운 연결 생성
- "아하!" 순간은 준비 후에 찾아옴

## Three B's의 의미

### Bed (침대)
- 수면 전후 반의식 상태
- 꿈에서 해결책 발견
- 아침 기상 직후 통찰

### Bath (목욕)
- 이완된 신체 상태
- 일상적 활동으로 의식 분산
- 물의 감각이 창의성 촉진

### Bus (버스/이동)
- 이동 중 단조로운 자극
- 환경 변화로 새로운 관점
- 반복적 움직임이 사고 자극

## 인큐베이션 프로세스
```
[준비] → [인큐베이션] → [통찰] → [검증]
집중적    문제에서       "아하!"   아이디어
문제      떠나기         순간      확인
분석
```

## 최적화 방법
1. **충분한 준비**: 문제를 깊이 이해한 후 휴식
2. **의도적 중단**: "이제 놓아준다" 의식적 결정
3. **다른 활동**: 산책, 운동, 취미 활동
4. **메모 준비**: 아이디어는 갑자기 옴
5. **시간 확보**: 최소 수 시간~며칠

## 통찰을 촉진하는 활동
- 샤워/목욕
- 산책
- 정원 가꾸기
- 운전 (익숙한 경로)
- 요리
- 명상

## 실제 사례
- 아르키메데스: 목욕 중 부력 발견
- 케쿨레: 꿈에서 벤젠 구조 발견
- 뉴턴: 사과나무 아래 휴식 중 중력 통찰

## 활용 상황
- 열심히 해도 답이 안 나올 때
- 복잡한 문제에 막혔을 때
- 창의적 도약이 필요할 때
