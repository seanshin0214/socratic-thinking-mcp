# Slice and Dice (Attribute Listing) - 속성 분해 기법

## 개요
문제나 대상을 구성 요소(속성)로 분해하여 각각을 개별적으로 분석하고 개선하는 기법입니다.

## 핵심 원리
- 복잡한 것을 단순한 부분으로 나눔
- 각 속성을 독립적으로 변형 가능
- 부분의 개선이 전체의 혁신으로 이어짐

## 적용 단계
1. **대상 선정**: 개선하고자 하는 제품/서비스/프로세스 선택
2. **속성 나열**: 모든 구성 요소와 특성을 나열
   - 물리적 속성: 크기, 색상, 재료, 형태
   - 기능적 속성: 용도, 작동 방식
   - 감각적 속성: 느낌, 소리, 냄새
3. **각 속성 분석**: 현재 상태와 가능한 변형 탐색
4. **변형 적용**: 각 속성에 변화를 주어 새로운 조합 생성
5. **최적 조합 선택**: 가장 효과적인 변형 조합 결정

## 예시: 우산 개선
| 속성 | 현재 | 변형 아이디어 |
|------|------|---------------|
| 크기 | 표준 | 접이식, 골프용 대형 |
| 재료 | 천 | 투명 비닐, 방수 나노 코팅 |
| 손잡이 | 곡선 | 접이식, LED 내장, 다기능 |
| 색상 | 단색 | UV 변색, 패턴 |

## 활용 상황
- 기존 제품 개선
- 새로운 제품 개발
- 프로세스 최적화

## 관련 기법
- 형태학적 분석
- SCAMPER
- 체크리스트 기법
