# Value Proposition Canvas (가치제안 캔버스)

## 개요
가치제안 캔버스는 고객 프로필과 가치맵을 매칭하여 제품-시장 적합성(Product-Market Fit)을 시각화하는 도구입니다. Business Model Canvas의 확장입니다.

## 1. 고객 프로필 (Customer Profile)

### Customer Jobs (고객 과업)
고객이 해결하려는 과업, 문제, 필요

#### 유형
```
기능적 과업 (Functional)
├── 특정 작업 수행
├── 문제 해결
└── 목표 달성

사회적 과업 (Social)
├── 좋게 보이고 싶음
├── 권력/지위 얻기
└── 소속감

감정적 과업 (Emotional)
├── 안심 느끼기
├── 두려움 줄이기
└── 기쁨 경험

지원 과업 (Supporting)
├── 구매자로서 (비교, 결정)
├── 공동창작자로서 (피드백)
└── 이전자로서 (폐기, 전환)
```

#### 질문
- 고객이 무엇을 달성하려 하는가?
- 어떤 과업을 완료하려 하는가?
- 어떤 필요를 충족시키려 하는가?

### Pains (고통)
과업 수행 전, 중, 후의 부정적 결과, 리스크, 장애물

#### 유형
```
원치 않는 결과/문제
├── 기능적: 작동 안 함, 부작용
├── 감정적: 기분 나쁨, 지루함
└── 부수적: 짜증나는 것들

장애물
├── 시작 못하게 하는 것
├── 느리게 하는 것
└── 완료 못하게 하는 것

리스크
├── 잘못될 수 있는 것
├── 중대한 부정적 결과
└── 예상치 못한 비용
```

#### 고통 강도 평가
```
극심함 ─────────────────── 경미함
  █████████████░░░░░░░
       이 고통이 얼마나 심각한가?
```

### Gains (이득)
고객이 원하거나 기대하는 결과와 혜택

#### 유형
```
필수 이득 (Required)
└── 없으면 솔루션이 작동 안 함

기대 이득 (Expected)
└── 기본적으로 기대하는 것

희망 이득 (Desired)
└── 기대 이상, 있으면 좋은 것

예상 밖 이득 (Unexpected)
└── 기대 못했지만 기뻐할 것
```

#### 질문
- 어떤 절약이 고객을 행복하게 하는가? (시간, 돈, 노력)
- 어떤 결과를 기대하는가?
- 어떤 것이 고객의 삶을 쉽게 만드는가?

## 2. 가치 맵 (Value Map)

### Products & Services (제품/서비스)
고객 과업 수행을 돕는 모든 제품과 서비스

#### 분류
```
물리적: 제조 상품
무형적: 저작권, 애프터서비스
디지털: 다운로드, 온라인 서비스
금융적: 펀드, 보험
```

### Pain Relievers (고통 해소자)
제품/서비스가 고객의 고통을 어떻게 해소하는지

#### 질문
- 시간, 돈, 노력을 절약해주는가?
- 기존 솔루션의 문제를 해결하는가?
- 어려움/도전을 제거하는가?
- 부정적 사회적 결과를 없애는가?
- 두려움/걱정을 줄이는가?
- 실수 가능성을 낮추는가?

### Gain Creators (이득 창출자)
제품/서비스가 고객의 이득을 어떻게 창출하는지

#### 질문
- 고객이 기대하거나 꿈꾸는 결과를 만드는가?
- 기존 솔루션을 능가하는가?
- 고객의 일/삶을 쉽게 만드는가?
- 긍정적 사회적 결과를 만드는가?
- 고객이 원하는 것을 해주는가?
- 비밀스러운 바람을 충족하는가?

## Fit (적합성) 평가

### 3가지 Fit 수준

```
Problem-Solution Fit
(문제-솔루션 적합성)
├── 해결할 가치 있는 문제 확인
├── 가설 검증
└── 인터뷰/관찰 기반

Product-Market Fit
(제품-시장 적합성)
├── 시장에서 실제 검증
├── 고객이 지불 의향
└── 성장 지표 확인

Business Model Fit
(비즈니스 모델 적합성)
├── 수익성 확보
├── 확장 가능
└── 지속 가능
```

### Fit 체크리스트

```
□ 모든 중요 고객 과업을 다루는가?
□ 가장 극심한 고통을 해소하는가?
□ 필수/기대 이득을 창출하는가?
□ 소수의 일을 탁월하게 하는가?
□ 경쟁 대비 차별화되는가?
□ 고객이 전환할 이유가 있는가?
```

## 작성 프로세스

### 단계 1: 고객 프로필 작성
1. 고객 세그먼트 선택
2. 인터뷰/관찰 데이터 수집
3. Customer Jobs 나열
4. Pains 나열
5. Gains 나열
6. 우선순위 지정 (중요도, 빈도)

### 단계 2: 가치 맵 작성
1. 제품/서비스 나열
2. Pain Relievers 매핑
3. Gain Creators 매핑
4. 강도 평가

### 단계 3: Fit 검증
1. 선 그어 연결
2. 미충족 영역 식별
3. 과잉 충족 영역 식별
4. 우선순위 재조정

## 예시: 온라인 영어 학습 앱

### 고객 프로필
```
Jobs:
- 영어로 비즈니스 미팅하기
- 해외 출장에서 소통하기
- 승진에 필요한 자격 갖추기

Pains:
- 학원 갈 시간 없음 (극심)
- 회화 실력 안 늘음 (극심)
- 창피해서 말 못함 (중간)
- 비용이 높음 (중간)

Gains:
- 언제 어디서나 학습 (필수)
- 실제 대화 연습 (기대)
- 진도 확인 (기대)
- 원어민과 대화 (희망)
```

### 가치 맵
```
Products/Services:
- AI 튜터 앱
- 음성 인식 발음 교정
- 1:1 원어민 화상 수업

Pain Relievers:
- 5분 단위 마이크로 러닝 → 시간 없음 해결
- AI 롤플레이 → 창피함 없이 연습
- 월 정액제 → 비용 부담 감소

Gain Creators:
- 모바일 앱 → 언제 어디서나
- 실전 비즈니스 시나리오 → 실제 대화 연습
- 성장 대시보드 → 진도 확인
- 원어민 예약 시스템 → 원어민 대화
```

## 활용 팁

### DO
- 고객 인터뷰로 검증
- 구체적이고 상세하게 작성
- 우선순위 명확히
- 정기적으로 업데이트

### DON'T
- 가정만으로 작성
- 모든 것을 해결하려 함
- 경쟁자 무시
- 한 번 작성 후 방치
