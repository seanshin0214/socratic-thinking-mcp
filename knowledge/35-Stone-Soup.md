# Stone Soup (Fantasy Questions) - 환상 질문 기법

## 개요
현실의 제약을 무시한 "만약에..." 질문을 통해 창의적 가능성을 탐구하는 기법입니다. 제약 없는 상상이 현실적 해결책의 씨앗이 됩니다.

## 핵심 원리
- 제약이 창의성을 제한
- 불가능을 상상하면 가능성이 보임
- 환상에서 현실로 역산

## 환상 질문 유형

### 마법 질문
- "마법 지팡이로 무엇을 바꾸겠는가?"
- "하룻밤 새 문제가 해결되었다면 무엇이 달라졌을까?"
- "무한한 자원이 있다면?"

### 시간 질문
- "시간을 멈출 수 있다면?"
- "10년 후에서 돌아본다면?"
- "과거로 돌아가 하나를 바꿀 수 있다면?"

### 능력 질문
- "투명인간이 될 수 있다면?"
- "마음을 읽을 수 있다면?"
- "순간이동이 가능하다면?"

### 규칙 질문
- "물리 법칙이 다르다면?"
- "돈이 존재하지 않는다면?"
- "실패가 불가능하다면?"

## 적용 프로세스
```
[제약 제거] → [자유 상상] → [핵심 추출] → [현실 적용]
   "만약..."    이상적      무엇이       어떻게
   질문         해결책      핵심인가?    가능하게?
```

## 예시
**문제**: 고객 서비스 개선

**환상 질문**: "고객의 마음을 읽을 수 있다면?"

**환상적 해결**: 고객이 말하기 전에 니즈 파악

**현실 적용**:
- 행동 데이터 분석으로 니즈 예측
- 선제적 서비스 제공
- AI 기반 추천 시스템

## 질문 템플릿
1. "만약 [제약]이 없다면 어떻게 할까?"
2. "만약 [불가능한 것]이 가능하다면?"
3. "완벽한 세상에서 이 문제는 어떻게 해결될까?"
4. "제약 없이 처음부터 설계한다면?"

## 활용 상황
- 관습적 사고에 갇혔을 때
- 획기적 혁신이 필요할 때
- 비전 수립
- 제품/서비스 재설계
