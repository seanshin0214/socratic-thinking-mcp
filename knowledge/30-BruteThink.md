# BruteThink (Random Stimulation) - 무작위 자극 기법

## 개요
무작위로 선택한 단어, 이미지, 물체를 문제와 강제로 연결하여 예상치 못한 아이디어를 생성하는 기법입니다.

## 핵심 원리
- 뇌는 패턴을 찾으려 함
- 무관한 요소의 강제 연결이 새로운 연상 유발
- 익숙한 사고 패턴 탈피

## 적용 단계
1. **문제 정의**: 해결하고자 하는 문제 명확히 기술
2. **무작위 자극 선택**:
   - 사전에서 무작위 단어 선택
   - 잡지에서 무작위 이미지 선택
   - 주변의 무작위 물체 선택
3. **속성 추출**: 선택한 자극의 특성, 기능, 연상 나열
4. **강제 연결**: 각 속성을 문제와 연결하여 아이디어 도출
5. **아이디어 발전**: 가능성 있는 아이디어 구체화

## 예시
**문제**: 직원 이직률 감소

**무작위 단어**: "나무"

**나무의 속성**:
- 뿌리가 깊다 → 회사에 대한 소속감 강화
- 계절마다 변한다 → 직원 성장 단계별 프로그램
- 가지가 뻗어나간다 → 경력 다양화 기회
- 그늘을 제공한다 → 멘토링 프로그램
- 물을 필요로 한다 → 정기적 피드백과 인정

## 무작위 자극 소스
- 사전/책
- 잡지/신문
- 주변 물체
- 온라인 무작위 단어 생성기
- 자연/동물
- 역사적 인물/사건

## 성공 비결
- 첫 번째 자극에 너무 집착하지 않기
- 처음에는 이상해 보여도 끝까지 연결 시도
- 양을 목표로 (질은 나중에 평가)
- 팀원들과 함께하면 더 풍부한 연상

## 활용 상황
- 창의적 막힘에 빠졌을 때
- 새로운 관점이 필요할 때
- 브레인스토밍 활성화
- 고정관념 탈피

## 관련 기법
- 강제 연결법
- 유추/비유 사고
- 시네틱스
