# False Faces (Reversal) - 역발상 기법

## 개요
기존 가정을 뒤집어 새로운 관점을 발견하는 사고 기법입니다. 당연하게 여기는 것들을 반대로 생각함으로써 혁신적인 아이디어를 창출합니다.

## 핵심 원리
- 모든 가정에는 반대 가정이 존재
- 뒤집힌 관점이 새로운 기회를 드러냄
- 관습적 사고의 틀을 깨는 방법

## 적용 단계
1. **가정 나열**: 문제에 대한 모든 가정을 적습니다
2. **각 가정 뒤집기**: 각 가정의 정반대를 작성합니다
3. **뒤집힌 가정 탐구**: 반대 가정이 사실이라면 어떻게 될지 상상합니다
4. **통찰 추출**: 새로운 관점에서 얻은 아이디어를 정리합니다

## 예시
- 가정: "고객이 매장을 방문한다" → 반전: "매장이 고객을 방문한다" → 이동식 판매, 배달 서비스
- 가정: "직원이 회사에 출근한다" → 반전: "직원이 집에서 일한다" → 재택근무

## 활용 상황
- 고정관념에 갇혀있을 때
- 기존 방식이 더 이상 효과가 없을 때
- 혁신적인 비즈니스 모델이 필요할 때

## 관련 기법

- 반대로 생각하기
