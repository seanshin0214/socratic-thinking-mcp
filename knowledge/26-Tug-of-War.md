# Tug-of-War (Force Field Analysis) - 힘의 장 분석

## 개요
변화를 촉진하는 힘(추진력)과 방해하는 힘(저항력)을 분석하여 효과적인 변화 전략을 수립하는 기법입니다. Kurt Lewin이 개발했습니다.

## 핵심 원리
- 모든 상황은 두 가지 힘의 균형 상태
- 추진력 강화 또는 저항력 약화로 변화 가능
- 저항력 제거가 추진력 강화보다 효과적

## 적용 단계
1. **목표 정의**: 원하는 변화 상태 명확히 설정
2. **현재 상태 파악**: 현재 위치와 목표 간 격차 인식
3. **추진력 나열**: 변화를 돕는 모든 요소 식별
4. **저항력 나열**: 변화를 방해하는 모든 요소 식별
5. **힘의 강도 평가**: 각 요소의 영향력 (1-5점)
6. **전략 수립**: 저항력 약화 + 추진력 강화 방안 도출

## 분석 다이어그램
```
추진력 (+)              목표              저항력 (-)
─────────►         【변화 달성】         ◄─────────

경영진 지원 ─────►                  ◄───── 직원 저항
(강도: 4)                                  (강도: 3)

비용 절감 ───────►                  ◄───── 기술 부족
(강도: 3)                                  (강도: 4)

시장 트렌드 ─────►                  ◄───── 예산 제약
(강도: 5)                                  (강도: 2)
```

## 전략 옵션
1. **저항력 제거**: 가장 강한 저항 요소 해소
2. **저항력 약화**: 저항 요소의 영향력 감소
3. **추진력 강화**: 지원 요소 확대
4. **새로운 추진력 추가**: 추가적인 지원 요소 발굴

- 협상 전략 수립
