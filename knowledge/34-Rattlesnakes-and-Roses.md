# Rattlesnakes and Roses (Analogies) - 유추 사고 기법

## 개요
서로 다른 영역 간의 유사성을 찾아 문제 해결에 적용하는 기법입니다. 익숙한 것에서 낯선 것을, 낯선 것에서 익숙한 것을 발견합니다.

## 핵심 원리
- 모든 것은 어떤 방식으로든 연결됨
- 한 영역의 해결책이 다른 영역에 적용 가능
- 창의성은 멀리 떨어진 것들의 연결

## 유추의 유형

### 직접 유추 (Direct Analogy)
다른 분야의 직접적 사례 적용
- 예: 새의 비행 → 비행기 설계
- 예: 도꼬마리 씨앗 → 벨크로

### 개인 유추 (Personal Analogy)
자신이 문제의 일부가 되어봄
- "내가 이 제품이라면 어떻게 느낄까?"
- "내가 고객이라면 무엇을 원할까?"

### 상징 유추 (Symbolic Analogy)
문제를 상징이나 은유로 표현
- 문제: 팀 갈등 → 상징: "불과 얼음"
- 해결: 중간 온도 (따뜻한 물) 찾기

### 환상 유추 (Fantasy Analogy)
마법이나 환상적 해결책 상상
- "마법 지팡이가 있다면?"
- "물리 법칙을 무시할 수 있다면?"

## 적용 단계
1. **문제 정의**: 핵심 문제 명확화
2. **유사 분야 탐색**: 비슷한 문제를 해결한 다른 분야 찾기
3. **유추 추출**: 그 분야의 해결 원리 파악
4. **원리 전이**: 원리를 현재 문제에 적용
5. **아이디어 구체화**: 실제 해결책으로 발전

## 예시
**문제**: 조직의 지식 공유 부족

**유추 분야**: 생태계
- 생태계의 양분 순환 → 지식 순환 시스템
- 곤충의 수분 → 지식 중개자 역할
- 균류의 네트워크 → 비공식 지식 네트워크

## 유추 소스
- 자연/생물학
- 역사적 사건
- 예술/음악
- 스포츠
- 요리
- 건축

## 활용 상황
- 기존 방식이 효과 없을 때
- 완전히 새로운 접근이 필요할 때
- 복잡한 문제 단순화
- 아이디어 설명/전달
