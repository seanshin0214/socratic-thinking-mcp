# Kipling Method (5W1H)

## Category
CRITICAL & SYSTEMS THINKING

## Description
Rudyard Kipling's "Six Honest Serving Men" - Who, What, Where, When, Why, and How. A comprehensive questioning framework that ensures no critical aspect is overlooked.

## When to Use
- Information gathering
- Problem definition
- Project planning
- Root cause analysis
- Situation assessment
- Journalism/reporting

## Question Sequence (6 questions)

### Question 1/6: WHAT?
What is happening? What is the problem/situation? What are the facts?

**Purpose:** Define the issue clearly

### Question 2/6: WHY?
Why is this happening? Why is it important? Why does it matter?

**Purpose:** Understand significance and causation

### Question 3/6: WHO?
Who is involved? Who is affected? Who is responsible? Who can help?

**Purpose:** Identify stakeholders and actors

### Question 4/6: WHERE?
Where is this happening? Where did it start? Where will it lead?

**Purpose:** Understand context and location

### Question 5/6: WHEN?
When did this start? When does it occur? When must action be taken?

**Purpose:** Establish timeline and urgency

### Question 6/6: HOW?
How did this happen? How does it work? How can we address it?

**Purpose:** Understand mechanism and potential solutions

## Expected Outcome
- Complete situation understanding
- No critical information gaps
- Clear problem definition
- Actionable next steps

## Academic Source
- Kipling, R. (1902). "Just So Stories" - "I Keep Six Honest Serving Men" poem
- Used in journalism, lean manufacturing, and problem-solving

## Tips
- Answer each question fully before moving on
- Each question should require more than yes/no
- Revisit questions as new information emerges
- Can be combined with 5 Whys for deeper analysis
