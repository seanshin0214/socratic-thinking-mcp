# OKR (Objectives and Key Results) - 목표와 핵심결과

## 개요
OKR은 야심찬 목표(Objectives)와 측정 가능한 핵심결과(Key Results)를 연결하여 조직의 집중력과 실행력을 높이는 목표 설정 프레임워크입니다. Intel의 Andy Grove가 개발하고 Google이 대중화했습니다.

## 핵심 구성요소

### Objective (목표)
- **정의**: 달성하고자 하는 질적 목표
- **특징**: 영감을 주고, 기억하기 쉽고, 야심차야 함
- **개수**: 분기당 3-5개가 적당

**좋은 Objective 예시**:
- "고객이 사랑하는 제품 만들기"
- "시장 리더십 확보"
- "세계 최고의 고객 서비스 구축"

**나쁜 Objective 예시**:
- "매출 20% 증가" (측정 가능 = KR)
- "더 열심히 일하기" (모호함)

### Key Results (핵심결과)
- **정의**: 목표 달성을 측정하는 정량적 지표
- **특징**: 구체적, 측정 가능, 시간 제한
- **개수**: 목표당 3-5개

**좋은 KR 예시**:
- "NPS 점수 40 → 60 달성"
- "월간 활성 사용자 100만 명 도달"
- "고객 응답 시간 24시간 → 2시간 단축"

**나쁜 KR 예시**:
- "고객 만족도 향상" (측정 불가)
- "열심히 노력함" (주관적)

## OKR 수립 프로세스

### 1단계: 조직 OKR 설정
```
회사 미션/비전
    ↓
연간 전략 목표
    ↓
분기 OKR
```

### 2단계: 팀 OKR 정렬
- 조직 OKR과 연결
- 팀의 기여 방식 명확화
- 50% 상향식 + 50% 하향식

### 3단계: 개인 OKR 설정 (선택)
- 팀 OKR에서 파생
- 개인 성장 목표 포함 가능

## OKR 운영 사이클

### 분기 시작 (1-2주차)
1. OKR 초안 작성
2. 팀/조직 정렬 미팅
3. OKR 최종 확정
4. 전사 공유

### 분기 중 (3-11주차)
1. **주간 체크인**: 진척도 업데이트 (10-15분)
2. **격주 팀 미팅**: 장애물 논의
3. **월간 리뷰**: 필요시 조정

### 분기 말 (12-13주차)
1. **점수 산정**: 0.0-1.0 스케일
2. **회고**: 배운 점, 개선점
3. **다음 분기 준비**

## OKR 점수 산정

### 점수 스케일
```
0.0-0.3: 실패 (목표 달성 못함)
0.4-0.6: 진전 있음 (부분 달성)
0.7-0.8: 성공 (목표 달성) ← 이상적 점수
0.9-1.0: 완벽 (목표가 쉬웠을 수 있음)
```

### 점수 해석
- **0.7이 이상적**: 야심찬 목표를 대부분 달성
- **1.0 반복**: 목표가 너무 쉬움 → 더 도전적으로
- **0.3 이하 반복**: 목표가 비현실적이거나 실행 문제

## OKR vs KPI

| 측면 | OKR | KPI |
|------|-----|-----|
| 목적 | 변화/혁신 추구 | 현상 유지/모니터링 |
| 성격 | 도전적, 야심찬 | 현실적, 달성 가능 |
| 주기 | 분기 | 연간/상시 |
| 달성률 | 70%가 성공 | 100% 기대 |
| 연동 | 보상과 분리 | 보상과 연동 가능 |

## OKR 작성 체크리스트

### Objective 체크리스트
- [ ] 영감을 주는가?
- [ ] 기억하기 쉬운가?
- [ ] 야심차지만 현실적인가?
- [ ] 조직/팀 미션과 연결되는가?
- [ ] 3-5개 이내인가?

### Key Results 체크리스트
- [ ] 정량적으로 측정 가능한가?
- [ ] 구체적인 숫자가 있는가?
- [ ] 기한이 명확한가?
- [ ] 결과 지향적인가? (활동 X)
- [ ] 달성 여부를 명확히 판단할 수 있는가?

## OKR 예시

### 회사 수준
**Objective**: 고객이 사랑하는 제품으로 시장을 리드한다

**Key Results**:
1. NPS 점수 45 → 65 달성
2. 월간 활성 사용자 50만 → 100만 명
3. 앱스토어 평점 4.2 → 4.7
4. 고객 이탈률 5% → 2%

### 팀 수준 (마케팅)
**Objective**: 브랜드 인지도를 폭발적으로 높인다

**Key Results**:
1. 소셜미디어 팔로워 10만 → 50만
2. 브랜드 검색량 200% 증가
3. 미디어 노출 50건 달성
4. 웹사이트 트래픽 100만 PV 달성

### 개인 수준 (마케터)
**Objective**: 콘텐츠 마케팅의 달인이 된다

**Key Results**:
1. 블로그 포스트 12개 발행
2. 바이럴 콘텐츠 3개 제작 (공유 1000+)
3. 이메일 구독자 5000명 확보
4. 콘텐츠 전환율 2% → 5%

## 흔한 실수와 해결책

### 실수 1: KR을 할 일 목록으로 만듦
- ❌ "블로그 10개 작성"
- ✅ "블로그를 통한 유입 1000명/월"

### 실수 2: 너무 많은 OKR
- ❌ 10개 Objective, 30개 KR
- ✅ 3-5개 Objective, 총 15개 이하 KR

### 실수 3: OKR과 보상 직접 연동
- ❌ OKR 달성 = 보너스
- ✅ OKR은 성장과 학습 도구

### 실수 4: 한 번 설정 후 방치
- ❌ 분기 초 설정, 분기 말 확인
- ✅ 주간 체크인, 월간 리뷰

## OKR 도입 로드맵

### Phase 1: 파일럿 (1-2분기)
- 리더십 팀만 OKR 적용
- 프로세스 학습
- 도구 선정

### Phase 2: 확산 (3-4분기)
- 전체 팀으로 확대
- 코치/챔피언 양성
- 문화 정착

### Phase 3: 최적화 (5분기+)
- 조직 맞춤 조정
- 지속적 개선
- 다른 시스템과 통합
